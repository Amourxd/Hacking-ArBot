☞ `API_ID` - Get your API_ID from <a href="https://my.telegram.org/">my.telegram.org</a><br>

☞ `BOT_USERNAME` - Username of your bot without '@'.<br>
☞ `BOT_ID` - User ID of your bot.<br>
☞ `OWNER_ID` - Your user ID as an integer.<br>
